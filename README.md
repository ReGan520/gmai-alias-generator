# Gmail 无限别名生成器 (Gmail Infinite Alias Generator)

一款极其精美、安全且强大的 **Gmail 无限别名生成工具**。采用纯前端单文件（`index.html`）架构，所有计算均在您的本地浏览器中完成，**零数据上传，完全保护隐私**。

通过利用 Gmail 邮箱的独特路由机制，本工具可以为您的一封主邮箱生成成百上千个不重复的接收别名。非常适合用于**网站注册、防追踪防定位、邮件分类过滤与拦截垃圾邮件**。

---

## ✨ 核心功能

1. **四种生成规则自由组合**
   - **点号变换 (Dot Trick)**：在用户名中任意插入点号。Gmail 会自动忽略点号（例如：`a.b.c@gmail.com` 与 `abc@gmail.com` 投递至同一邮箱）。
   - **大小写变换 (Case Trick)**：随机改变英文字母的大小写。Gmail 识别邮箱时不区分大小写，但许多第三方网站会视为不同邮箱。
   - **加号魔法 (Plus Trick)**：在用户名后附加 `+` 及后缀（例如：`abc+netflix@gmail.com`）。
     - *加号增强子模式*：支持**自定义文本**、**递增数字**（防重复）、**随机字符**（防追踪）和**当前日期**。
   - **域名变换 (Domain Trick)**：将 `@gmail.com` 替换为 `@googlemail.com`。两者在 Google 邮件系统内完全互通。

2. **批量与去重控制**
   - 支持一键选择生成数量：`1`、`5`、`10`、`20`、`50`、`100` 个。
   - 内置**智能去重算法**，自动剥离输入邮箱的多余修饰，解析出最基础的 Clean 用户名。
   - **防死循环安全保护**：当用户名过短且规则限制导致无法生成足量不重复邮箱时，系统会自动截断输出并弹出友好警告，避免页面卡死。

3. **极佳的交互与导出**
   - **单项复制**：一键复制单个别名，带对勾及 Toast 提示反馈。
   - **批量复制**：一键复制当前列表中所有显示的邮箱别名。
   - **导出为 TXT**：一键下载 `.txt` 别名列表文件，方便导入其他平台。
   - **实时搜索过滤**：在生成的大量别名中输入关键字，瞬间进行模糊筛选。
